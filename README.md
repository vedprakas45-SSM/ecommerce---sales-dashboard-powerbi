📊 Ecommerce Sales Dashboard (Power BI)

🔍 Overview

This project is an interactive Power BI dashboard built to analyze ecommerce sales data.
It provides insights into sales performance, customer trends, and country-wise analysis.

---

📸 Dashboard Preview

"Dashboard" (powerbi-dashboard.png)

---

🛠️ Tools Used

- Power BI
- SQL
- Excel

---

📊 Key Features

- Total Sales KPI (1.18M)
- Top 10 Customers by Sales
- Sales by Country
- Sales Trend Over Time
- Interactive Filters (Date & Country)

---

🧠 Insights

- Identified top-performing customers contributing highest revenue
- Country-wise analysis shows which regions drive maximum sales
- Sales trend highlights peak months and performance fluctuations

---

📁 Files in Repository

- "sales dashboard powerbi 6.pbix" → Power BI Dashboard file
- "powerbi-dashboard.png" → Dashboard screenshot

---

🚀 How to Use

1. Download the ".pbix" file
2. Open in Power BI Desktop
3. Explore dashboard with filters and visuals

---

📌 Project Objective

To build a clean and interactive dashboard that helps in understanding ecommerce sales performance and supports data-driven decision making.
